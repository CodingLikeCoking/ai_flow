# Codex and Cursor MCP Setup

## Codex

1. Run `ai-flow print codex-mcp-config`.
2. Copy the printed JSON snippet into your Codex MCP configuration.
3. Ensure the command path resolves to `ai-flow-mcp`.
4. Restart the client and verify `ai_flow_list_projects` appears.

## Cursor

1. Run `ai-flow print cursor-mcp-config`.
2. Copy the printed JSON snippet into Cursor's MCP configuration.
3. Restart Cursor.
4. Verify the `ai-flow` server is connected and the tools are discoverable.
